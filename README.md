Personal Website vreshch.com, contains CV, interests, Contacts;

## Technologies Used

* Rect & Typescript
* Deployed to [Azure Static Web Apps](https://azure.microsoft.com/en-us/services/app-service/static/)

## Quick Start

* Start local development

```bash
npm run dev
```

* Build project

```bash
npm run build
```

* Start project

```bash
npm start
```

* Run linter

```bash
npm run lint
```
